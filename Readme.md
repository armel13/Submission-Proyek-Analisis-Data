# Proyek Analisis Data: E-Commerce Public Dataset

## Deskripsi Proyek
Proyek ini bertujuan untuk menganalisis data e-commerce publik dari Brasil. Data yang digunakan terdiri dari dua dataset: `products_dataset.csv` dan `sellers_dataset.csv`. Analisis dilakukan untuk menjawab dua pertanyaan bisnis:
1. Negara bagian mana yang memiliki konsentrasi penjual di atas 10% dari total penjual, dan bagaimana distribusi penjual di tiga wilayah utama (SP, PR, MG)?
2. Kategori produk apa saja yang memiliki rata-rata berat di atas 5.000 gram, dan berapa persentase kontribusi produk-produk tersebut terhadap total keseluruhan produk?

Selain itu, dilakukan juga analisis lanjutan seperti binning berat produk menjadi tiga kategori (ringan, sedang, berat) dan melihat proporsinya pada 10 kategori produk teratas.
